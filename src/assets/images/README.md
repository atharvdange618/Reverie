# Reverie Images

This directory contains images and icons for the app.

## Structure

```
images/
├── icon/           # App icons
├── splash/         # Splash screen assets
└── illustrations/  # UI illustrations
```

## App Icon

The app icon should embody the "Reverie" concept:

- Dreamy, soft aesthetic
- Possibly a book, moon, or abstract shapes
- Works well in both light and dark contexts
