# Reverie Fonts

This directory contains the custom fonts used in Reverie.

## Required Fonts

### Literata (Reading Font)

Download from Google Fonts: https://fonts.google.com/specimen/Literata

Place these files here:

- `Literata-Regular.ttf`
- `Literata-Medium.ttf`
- `Literata-SemiBold.ttf`
- `Literata-Italic.ttf`

### Inter (UI Font)

Download from Google Fonts: https://fonts.google.com/specimen/Inter

Place these files here:

- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

## Installation

After placing the font files, run:

```bash
npx react-native-asset
```

This will link the fonts to the native projects.
