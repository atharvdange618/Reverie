# Reverie Audio

This directory contains ambient audio tracks for the reading experience.

## Default Tracks

Place your ambient audio files here (MP3 format only):

Example:

- `ambient-piano.mp3`
- `soft-lofi.mp3`
- `rain-sounds.mp3`

## Audio Guidelines

- Keep tracks at a reasonable file size (under 10MB recommended)
- Use calm, non-intrusive music
- Looping-friendly tracks work best
- Soft piano, lofi, nature sounds are ideal
